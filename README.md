# Telegram bot for self financies on Laravel

## Setup
required: docker

1. > cp .env.dist .env
2. Insert values in .env required:
+ TELEGRAM_BOT_TOKEN
4. Use docker-compose up

## Branches rules
- feature - Branches of new features
- fix - Branches fixes
- hotfix - Branches for hotfixes
- refactor - Branches of refactor

Example: feature/import-users-from-social

## Commits message format
Example: feat: maked new job for importing

type:
    - feat: description about a feature
    - fix: description about a fix
